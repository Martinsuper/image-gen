# 灰度 PNG 转换器

一个跨平台 Web/PWA 小工具，用浏览器本地能力把照片转换为 PNG 灰度图片。图片不会上传到服务器。

## 功能

- 拖拽或选择多张图片
- 原图和灰度结果并排预览
- 标准亮度、平均值、去饱和三种灰度算法
- 可调整灰度强度、亮度、对比度、Gamma、黑场、白场
- 支持自动色阶、反相和灰阶层次控制
- 支持有序点阵和误差扩散抖动
- 可保留透明通道
- 导出当前图片，或将批量结果打包为 ZIP
- 支持 PNG / SVG 嵌入图 / SVG 路径描摹
- SVG 路径描摹在 Web Worker 中运行，减少页面卡顿
- 支持 PWA 离线缓存

## 关于 SVG 导出

`SVG 嵌入图` 会把处理后的灰度结果封装进 SVG 文件，适合需要 SVG 容器、跨平台嵌入或继续排版的场景，照片本身仍然是像素图。

`SVG 路径描摹` 会使用 ImageTracerJS 把灰度像素描摹成 SVG path，是真正的矢量输出。照片类图片会呈现为海报化色块效果，文件大小和路径复杂度会随图片细节增加。

## 本地运行

```bash
python3 -m http.server 5173
```

然后打开：

```text
http://localhost:5173
```

## 后续桌面化

如果需要桌面安装包，可以用 Tauri 把这个静态目录作为前端资源封装。核心图片处理逻辑都在 `app.js`，后续迁移成本很低。

## Cloudflare Pages

推荐配置：

- 构建命令：`npm run build`
- 构建输出目录：`dist`

仓库里也提供了 `wrangler.toml`，明确把 Pages 输出目录和 Workers 静态资源目录都指向 `dist`，避免 Cloudflare 把仓库根目录或 `node_modules` 当作静态资源上传。
